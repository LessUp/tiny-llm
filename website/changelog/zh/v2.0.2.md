# v2.0.2

**发布日期:** 2026-04-27

## 概要

代码质量改进、GGUF 量化工具和仓库加固。

## 变更内容

### 代码质量

- **KVCacheManager**: 将构造函数改为工厂方法 `create()`，实现一致的 `Result<T>` 错误处理
- **noexcept**: 为简单访问器方法添加 `noexcept` 说明符
- **F32→F16 转换**: 提取为工具函数 `convertF32ToF16()`，位于新的 `quantization.h`
- **代码格式**: 修复 quantization.cpp、kv_cache.cpp、model_loader.cpp 的 clang-format-18 违规
- **main.cpp**: 新增 `--help`、`--version`、`--info` 命令行选项，提升易用性

### 基础设施

- **CI 简化**: 使用 `Jimver/cuda-toolkit` action 替代复杂的 CUDA fallback 逻辑
- **LSP 配置**: 增强 `.clangd` 配置，添加 fallback 编译标志以改善 IDE 支持
- **AI 工具链**: 在 CLAUDE.md 和 AGENTS.md 中添加核心架构文档

### 新功能

- **量化工具**: 新增 `quantization.h`/`quantization.cpp`，包含：
  - `convertF32ToF16()` - F32 到 F16 转换
  - `dequantizeQ4_0()` - Q4_0 块反量化
  - `dequantizeQ8_0()` - Q8_0 块反量化
  - `quantizeF16ToW8A16()` - F16 到 W8A16 量化

### 仓库清理

- 删除空的 `docs/` 目录
- 删除弃用的 `_bmad/` 和 `_bmad-output/` 目录
- 删除 `.qwen/` 配置
- 归档 `normalize-archive-readiness` 变更
- 精简 `.claude/skills/` 从 46 个技能（238 个文件）到 5 个核心技能

### 文档

- 更新版权年份到 2026
- 更新 API 文档中的 KVCacheManager 使用示例
- 添加 v2.0.2 的中英文变更日志
- 在 CLAUDE.md 和 AGENTS.md 中添加核心架构表

## 兼容性

- 无破坏性 API 变更
- `KVCacheManager` 构造函数现为私有；请使用 `KVCacheManager::create()` 工厂方法

## 升级指南

```cpp
// 旧版本 (v2.0.1)
KVCacheManager cache(config);

// 新版本 (v2.0.2)
auto cache_result = KVCacheManager::create(config);
if (cache_result.isErr()) {
    // 处理错误
}
auto cache = std::move(cache_result.value());
```
