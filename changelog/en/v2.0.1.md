## Fixed

### QuantizedWeight Scale Dimension Calculation

**Severity**: Critical  
**File**: `tests/test_integration.cu`

The `createRandomWeight` function had incorrect scale tensor dimension calculation.

```cpp
// Incorrect
int num_groups = (cols + group_size - 1) / group_size;
w.scales = randomDeviceFP16(rows * num_groups, ...);

// Correct  
int num_groups = (rows + group_size - 1) / group_size;
w.scales = randomDeviceFP16(num_groups * cols, ...);
```

**Impact**: W8A16 matmul uses `[rows/group_size, cols]` to index scales, requiring `ceil(rows/g) * cols` elements.

### Attention Kernel Cleanup

**File**: `kernels/attention.cu`

Removed 12 lines of unused `q_reg` array loading code in `attention_decode_kernel`.
